🛠️ How to Connect Open Dental

1
Install Open Dental on your practice computer if you haven't already. Download from opendental.com
2
Enable the Open Dental API — In Open Dental go to Setup → API → Enable REST API. Set the port to 8080.
3
Get your API key — In Open Dental go to Setup → API → Developer Keys → Create new key. Copy the key.
4
Add your API key to the opendental.py file. Replace your_api_key_here with your actual key.
OPENDENTAL_API_KEY = "your_actual_key_here"
5
Restart AI Insurance Guardian and come back to this page. You should see a green connected status.
